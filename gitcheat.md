#Git cheatsheet
**Name:** Aleksandr Segal
**Date:** February 15, 2017
    git config --global user.name "[name]
    git config --global user.email "[email address]
    git config --list
    git init
    git remote add origin [URL of your remote repo]
    git remote -v

    git status
    git add
    git commit -m "[message]"
